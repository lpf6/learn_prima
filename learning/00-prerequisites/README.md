# 第零阶段：前置知识

## 概述

本阶段提供学习 Prima.cpp 所需的基础知识，包括 C++ 编程、并行计算和深度学习基础。如果你已经具备相关经验，可以跳过本阶段。

## 能力目标

完成本阶段后，你将能够：

### 能做什么

| 能力 | 具体表现 | 相关代码 |
|------|----------|----------|
| **理解 C++ 代码** | 阅读模板、内存管理代码 | 项目源码 |
| **理解并行概念** | 理解数据并行、任务并行 | 内核设计 |
| **理解神经网络** | 理解注意力机制、Transformer | 模型架构 |

### 还不能做什么

- 编写高性能 C++ 代码
- 设计并行算法
- 实现神经网络架构

### 实际工作示例

学完本阶段后，你可以：

1. **理解 C++ 模板代码**
```cpp
// 你能理解这段模板代码
template<typename T, int BLOCK_SIZE>
__global__ void kernel(T *data, int n) {
    T shared_data[BLOCK_SIZE];
    // ...
}
```

2. **理解并行加速比**
```cpp
// 你能理解 Amdahl 定律
// 加速比 = 1 / (S + P/N)
// S: 串行部分比例，P: 并行部分比例，N: 处理器数
```

3. **理解 Transformer 架构**
```
你能理解 Transformer 的基本结构：
Input → Embedding → Positional Encoding → 
[Attention + FFN] × N → Output
```
