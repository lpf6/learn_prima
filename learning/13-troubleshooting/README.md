## 快速诊断表

| 问题类型 | 症状 | 可能原因 | 排查工具 |
|---------|------|---------|---------|
| **编译失败** | 编译错误 | 依赖缺失、语法错误 | 编译器输出 |
| **CUDA 错误** | kernel 启动失败 | 显存不足、配置错误 | cuda-memcheck |
| **性能低下** | 速度慢 | 内存瓶颈、占用率低 | Nsight Compute |
| **结果错误** | 输出不正确 | 算法错误、精度问题 | 单元测试 |
| **崩溃** | Segfault | 空指针、越界访问 | GDB、ASAN |
| **死锁** | 程序挂起 | 同步问题、资源竞争 | 线程分析 |

## 常用命令速查

### CUDA 相关

```bash
# 检查 GPU 状态
nvidia-smi

# 检查 CUDA 版本
nvcc --version

# 内存检查
cuda-memcheck ./program

# 性能分析
nsys profile ./program
```

### 系统相关

```bash
# 查看系统信息
uname -a
lscpu
free -h

# 查看进程
ps aux | grep program
top

# 查看日志
dmesg | tail
journalctl -xe
```

### Git 相关

```bash
# 查看变更
git diff
git log --oneline

# 回退版本
git checkout <commit>
git revert <commit>
```
